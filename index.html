I came up with Easy Privacy Node after dealing with introducing security software to non technical relatives and friends:
  • Most people have broadband internet access at home and usually run some kind of PC continuously connected to the Internet
  • Security software is usually over complex. This leads to a poor adoption rate. E.g. the adoption rate of PGP is very low so it usually makes no sense sending PGP encrypted emails.
  • Setting up your own secure VPN or VOIP server is quite a challenge even for experienced users.
  • Using encrypted Voice Over IP is practically impossible because non of the popular providers is supporting it.
  • Many providers try to lock you into their cloud platform loosing control over your data. In general if unencrypted is stored in a cloud service, then misuse is always possible.
The idea is to offer an easy to run software that can be downloaded and run within minutes on many platforms like your home PC, a small computer (e.g. Raspberry Pi) or even your trusted cloud service.
  • You are in control of your private data and you can decide what to publish to others. We make it easy that you become independent from providers like Google, Facebook, Dropbox etc. and other proprietary software platforms. Privacy violations will become an issue of the past.
  • You are able to access emails, documents etc. from anywhere though your Web browser in a secure way.
  • There is an easy integration with your mobile devices.
  • You are able to communicate securely with others even if they do not have any security software installed.

FAQ

On which software Easy Price Node runs on?
A Linux distribution with
  • Encrypted EMail: Postfix server with SSL and PGP support, IMAP/POP3 over SSL.
  • Encrypted Voice Over IP: Asterisk Voice Over IP Server with encryption.
  • Safe Web browsing: OpenSSL, OpenVPN and Tor server.
  • Accessible IP address: DynDNS
  • Document sharing: SFTP/SCP.
  • Orchestrating all services: Node.js
  • Everything is kept transparent: source code published via Github
  • Automatic update: using the Debian package system
What does Easy Privacy Node not offer?
  • We do not offer cloud based services. This does not prevent a user from running an Easy Privacy Node in a cloud. Our goal is getting private data back under the control of the user and reestablishing privacy.
  • We do not sell hardware devices. We concentrate on the software making privacy easily accessible to our customers. We support most platforms.
  • We are not proprietary. We utilize open source and standard protocols. Everything is transparent. Advanced user are able to customized or enhance their Easy Privacy Nodes.
What are future enhancements?
Possible enhancements
  • Cached blog/web sites: support of high bandwidth blogs/web sites in a Cloud fed via the local Easy Privacy Node
  • Easy Privacy Node Communities: allow the aggregation of communities (e.g. a family or a group of friends sharing information)
  • Maintenance of software and regular updates of Easy Privacy Nodes (similar to Windows Updates)
  • In the future there may be more powerful routers or mobile phones running an Easy Privacy Node native or in a virtual machine.
